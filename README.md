## 🎯 操作指南

### 抽卡
- `F` 快捷键单抽，`R` 快捷键十连
- 单抽：九宫格中心显示一个武将
- 十连：九个格子全部填满，一次性看全结果

### 阵容编辑
1. 点击右侧武将列表选中一个武将
2. 点击左侧九宫格空位 → 放上去
3. 点击九宫格已有武将格子 → 移除这个武将
4. 调整完点击「保存阵容」
5. 点击「清空阵容」清空所有格子

### 挑战战斗
- 进去就能看到你和敌方完整九宫格阵容
- 关闭「自动战斗」 → 点「下一步」一步步打
- 打开「自动战斗」 → 自动打完直接出结果
- 滑块调速度，越快自动战斗越快
- 胜利获得声望奖励，提升排名
