## 核心玩法

1.  **挂机** → 上阵武将自动产出国运点
2.  **招募** → 消耗国运点随机抽武将
3.  **阵容** → 搭配势力，激活羁绊，提升战力
4.  **挑战** → 挑战排行榜AI，提升排名拿奖励
5.  **循环** → 收集更多武将，更好搭配，更高战力

- ESC 或者右下角返回主页

# 《国运：三国招贤阁》完整策划案补充  
## ——区域经营与攻城略地（整合版）

本补充案在原有“三国招贤阁”基础上，引入**区域攻略**作为游戏主线，将单一的攻城进度拆分为多个可逐步征服的区域，并配合**武将/文臣分工**、**钱粮兵资源**、**随机事件**和**主动策略**，构建一个目标明确、层次丰富的挂机经营体验。

---

## 一、核心玩法调整

| 新增要素 | 融合方式 |
|----------|----------|
| **区域攻略** | 游戏主线：玩家需依次征服多个区域（州郡），每个区域有独立进度条和特性 |
| **武将/文臣分工** | 武将（勇武）主攻城速度，文臣（智略）主资源产出；可切换**上阵/镇守**状态 |
| **钱、粮、兵资源** | 新增三种资源，用于维持攻城、触发事件、主动策略；与原有聚贤令、威名并列 |
| **随机事件** | 每小时概率触发，事件库与当前区域绑定，受武将技能和联动影响 |
| **主动消耗（策略）** | 消耗钱粮兵获取临时增益，增加策略选择 |
| **Bongo Cat调整** | 敲击不再加速挂机，改为**额外产出聚贤令**，保持互动乐趣 |

## 二、区域攻略系统

### 2.1 区域划分（示例）
| 区域 | 解锁条件 | 特性 | 专属事件示例 | 首次通关奖励 |
|------|----------|------|--------------|--------------|
| 中原 | 初始开放 | 基础消耗均衡 | 官渡之战、商人来访 | 随机蓝色武将 |
| 江东 | 中原通关 | 粮耗+20% | 水贼、周瑜拜访 | 周瑜/陆逊碎片 |
| 西凉 | 江东通关 | 兵耗+20%，速度-10% | 马匪、匈奴入侵 | 马超/庞德碎片 |
| 巴蜀 | 西凉通关 | 速度-20%，事件概率+10% | 蜀道难、诸葛亮相助 | 诸葛亮/法正碎片 |
| 荆州 | 巴蜀通关 | 四战之地，事件频率+20% | 赤壁之战复刻 | 关羽/张飞碎片 |

### 2.2 区域攻略机制
- **进度条**：每个区域拥有独立进度条（0→100%），由**上阵武将总勇武**和**上阵文臣总智略**共同决定推进速度。
- **资源消耗**：每推进一定比例（如5%），自动扣除粮和兵。消耗量受区域特性影响（如江东粮耗+20%）。
- **区域解锁**：当前区域进度达100%后自动解锁下一个区域，同时弹出通关奖励。
- **多区域并行**（后期）：可解锁多个区域同时攻略，需分配上阵队伍（MVP阶段保持线性）。

### 2.3 区域事件系统
- **事件池分离**：每个区域拥有独立的事件池（5-8个专属事件），区域解锁后其事件加入全局随机池。
- **进度触发事件**：当区域进度达到25%、50%、75%时，强制触发一个区域特色事件（如“中原·官渡之战”），给予丰厚奖励。
- **事件影响**：某些事件结果会改变当前区域后续的消耗倍率或事件概率（例如成功赈灾使该区域粮耗-10%）。

### 2.4 区域奖励与成长
- **首次通关奖励**：每个区域首次征服时，给予武将碎片、资源包、宝物或永久增益（如“中原征服：钱粮产出+5%”）。
- **镇守槽位解锁**：部分区域通关后可增加**镇守槽位**，使更多武将/文臣在后方产出资源。
- **隐藏区域**：满足特定条件（如用特定羁绊通关）可解锁隐藏区域（如“南中”、“辽东”），增加探索感。

## 三、武将/文臣分工与状态

### 3.1 属性分工
- **武将**：勇武（影响攻城速度）、武力（影响战斗事件）
- **文臣**：智略（影响攻城速度）、政治（影响资源产出）
- 原有魅力、智力可转化为事件相关属性（如魅力高的事件成功率增加）

### 3.2 上阵/镇守状态
- **上阵**：武将/文臣在前线，100%发挥攻城能力；但不参与资源产出。上阵总人数有上限（初始3人，随招贤阁等级提升）。
- **镇守**：在后方，按比例（如50%）贡献能力用于资源产出：武将产兵，文臣产钱和粮。镇守槽位数量随区域解锁增加（初始1个，每征服一个区域+1）。
- **配置界面**：在阵容编辑中，玩家可为每个已拥有的武将/文臣选择状态（上阵/镇守）。

---

## 四、资源系统

### 4.1 资源类型
- **聚贤令**：用于招募（原有）
- **钱**：用于事件、主动策略、购买特殊物品
- **粮**：维持攻城消耗，也用于事件
- **兵**：攻城消耗主力，也用于事件

### 4.2 产出方式
- **基础产出**：根据镇守武将/文臣能力自动产出（例如：每镇守1点勇武，每秒产兵+0.1；每点智略，产钱+0.05、粮+0.05）。
- **事件产出**：随机事件可能给予资源。
- **后台敲击**：Bongo Cat敲击额外产聚贤令（每次1-3个，暴走5-10个）。
- **主动策略**（见第六节）。

### 4.3 消耗方式
- **攻城自动消耗**：每推进5%进度，扣除一定粮、兵（计算公式：基础消耗 × 区域倍率）。
- **事件选项消耗**。
- **主动策略消耗**。

---

## 五、随机事件系统

### 5.1 触发机制
- 每小时（游戏时间）有概率触发，基础概率10%。概率受镇守文臣智略影响（每10点智略+1%），受羁绊影响（如“卧龙凤雏”使事件概率翻倍）。
- 事件触发时弹出窗口，显示事件描述和2-3个选项。

### 5.2 事件类型（示例）
- **战斗事件**：敌军来袭，可选迎战（消耗兵）或避战（损失进度）。
- **天灾**：粮草被劫，可选赈灾（消耗钱）或硬扛（损失粮）。
- **商人**：可用钱低价购买粮或兵。
- **武将论道**：根据上阵武将组合，获得临时增益（如攻城速度+20%持续1小时）。
- **区域特色事件**：如江东的“水贼劫掠”、西凉的“马匪横行”等。

### 5.3 技能影响
部分武将/文臣拥有特殊技能，可改变事件选项或结果，例如：
- 诸葛亮：事件成功率+20%
- 张飞：战斗事件中，若选择迎战，额外获得进度
- 貂蝉：商人事件中，购买价格降低20%
- 华佗：天灾事件中，可消耗少量粮恢复部分兵力

### 5.4 联动事件
特定组合触发专属事件，获得稀有奖励：
- 五虎上将：触发“五虎破敌”，获得大量进度
- 卧龙凤雏：触发“隆中对”，永久提升智略
- 江东双璧：触发“赤壁之火”，获得特殊宝物
