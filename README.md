# 三国名将抽卡布阵游戏

> 基于 Godot 4.x 开发的三国抽卡+九宫格布阵+PVP战斗游戏

## 🎯 游戏玩法

1. **抽卡**：消耗国运点抽取三国名将，支持单抽(F快捷键)和十连抽(R快捷键，九折优惠)
2. **布阵**：在左侧 3x3 九宫格放置你拥有的武将，调整阵容
3. **PVP挑战**：匹配对手，回合制战斗，胜利提升排名获得声望奖励
4. **收集**：重复武将自动转换成碎片，可以用来兑换高级武将

## 📊 概率设定

| 稀有度 | 概率 | 碎片转换 | 边框颜色 |
