## 核心玩法

1.  **挂机** → 上阵武将自动产出国运点
2.  **招募** → 消耗国运点随机抽武将
3.  **阵容** → 搭配势力，激活羁绊，提升战力
4.  **挑战** → 挑战排行榜AI，提升排名拿奖励
5.  **循环** → 收集更多武将，更好搭配，更高战力

## 项目结构

```
scripts/
├── autoload/          # 自动加载单例（其实已经在project.godot配置了autoload）
├── main.gd            # 主界面逻辑
├── recruit.gd         # 招募界面逻辑

├── recruit.tscn       # 招募界面
├── lineup.tscn        # 阵容编辑界面
└── library.tscn       # 武将库界面

data/
└── heroes.json        # 武将数据表
```

## 如何运行

使用Godot 4.x 打开项目，点击运行即可。

- 主界面点击「招募」进入招募
- 点击「招募武将」按钮抽卡，按F可以快速抽
- 点击「阵容」进入阵容编辑
- 点击武将 → 点击空位 → 上阵
- ESC 或者右下角返回主页
